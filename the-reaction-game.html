The Reaction Game

The reaction game is a simple game made for mobile devices. The aim of the game is to react to the green screen that pops up before it disappears. The game is unfinished, as there is no game over screen yet, but if the player reacts when the green screen is not on then a game over should initiated. The game will be on the Google Play Store in the near future.
